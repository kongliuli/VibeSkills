avalonia-ui-optimizer（Vibe skill 元包）

此目录包含最小化技能元信息与一个平台检查模板，便于 Vibe 在导入时展示与识别。

使用说明（简要）：

上传到仓库路径： Skills/C%23/Avalonia/Avalonia-UI-Optimizer/
文件：
SKILL.md（必需，元信息）
templates/platform-checklist.txt（可选模板，供展示）
若需生成 ZIP：在本地把本目录压缩为 .skill 或 .zip，并将 SKILL.md 放在压缩包根目录。
